Visual SourceSafe
-C オプション (コマンド ライン)
参照 
 






コマンドにコメントを指定します。-C オプションを指定しないと、Visual SourceSafe は、コマンド ラインで指定した項目ごとにコメントを要求します。このオプションを指定すると、すべての項目が同じコメントを受け取ります。

次の表に -C オプションの使用方法を示します。

-C

すべての項目が同じコメントを受け取るように指定します。

-C-

コマンドにコメントを使用しないように指定します。

-Ctext

すべての項目のコメントとして文字列テキストを使用します。

-C@<ファイル名>

指定したファイルの内容を、すべての項目のコメントとして使用します。

-C?

Visual SourceSafe の既定の動作に戻り、個々のファイルごとにコメントを要求します。

解説

上の表に記載したオプションのいずれかを Checkout コマンドの既定値として設定するには、Ss.ini ファイルの Checkout_Comment 初期化変数を使用します。Ss.ini ファイルで既定値を設定している場合でも、特定のコマンドに対して -C? オプションを使用すると、既定値を置き換えることができます。また、Ss.ini ファイルで Comment_Editor 変数を設定すると、Visual SourceSafe が使用するプロンプト方式のコメント入力の代わりに、選択したエディタを呼び出すことができます。
